== Installation ==

1. Copy the `acf-polylang-translatable-field` folder into your `wp-content/plugins` folder
2. Activate the plugin via the plugins admin page
3. Create a new field via ACF and select the Polylang Translatable type
4. Read the description above for usage instructions
